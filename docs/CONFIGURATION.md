## 配置方式

PBD 和 RVO 参数通过场景中的 **Skelot PBD Plane** Actor 配置。

### 使用方法

1. 在场景中放置 `Skelot PBD Plane` Actor
2. 在详情面板中配置 PBD 和 RVO 参数
3. 参数会在运行时自动应用到 SkelotWorld

### 障碍物

使用以下 Actor 添加障碍物：
- `Skelot Sphere Obstacle` - 球形障碍物
- `Skelot Box Obstacle` - 盒形障碍物

## PBD 碰撞参数

PBD (Position Based Dynamics) 碰撞系统用于实例间的碰撞检测和响应。

### 基础参数

| 参数 | 类型 | 默认值 | 范围 | 说明 |
|------|------|--------|------|------|
| bEnablePBD | bool | true | - | 启用PBD碰撞 |
| CollisionRadius | float | 60 | 10-200 | PBD碰撞半径（厘米） |
| IterationCount | int32 | 3 | 1-10 | PBD迭代次数 |
| PostObstacleIterations | int32 | 1 | 0-5 | 障碍物碰撞后额外迭代次数 |
| RelaxationFactor | float | 0.3 | 0.1-1.0 | 松弛系数，控制约束响应强度 |
| GridSizeMultiplier | float | 1.0 | 0.5-2.0 | PBD网格尺寸倍率 |
| MaxNeighbors | int32 | 64 | 16-256 | 最大邻居数量 |
| UpdateFrequency | int32 | 1 | 1-4 | 更新频率（每N帧更新一次） |

### 速度投影参数

| 参数 | 类型 | 默认值 | 范围 | 说明 |
|------|------|--------|------|------|
| bEnableVelocityProjection | bool | true | - | 启用速度投影，防止穿透后抖动 |
| VelocityProjectionStrength | float | 0.8 | 0.0-1.0 | 速度投影强度，越高越稳定 |
| VelocityRecoveryRate | float | 2.0 | 0.5-5.0 | 碰撞后恢复原速度的速率 |
| MaxPushForceFactor | float | 1.5 | 1.0-3.0 | 深度重叠时的推力上限系数 |

### 参数详解

#### CollisionRadius（碰撞半径）

- **作用**: 定义每个实例的碰撞球体半径
- **影响**:
  - 过小：实例会重叠穿插
  - 过大：实例间距过大，密度降低
- **建议**: 根据角色体型调整，通常为角色宽度的一半

#### IterationCount（迭代次数）

- **作用**: 约束求解的迭代次数
- **影响**:
  - 过低：碰撞响应不准确，可能穿透
  - 过高：性能开销增加
- **建议**: 2-4 次，复杂场景可增加到 5 次

#### RelaxationFactor（松弛系数）

- **作用**: 控制每次迭代的位置校正强度
- **影响**:
  - 过低（0.1-0.3）：响应慢，但稳定
  - 过高（0.6-1.0）：响应快，但可能抖动
- **建议**: 0.3-0.5，高密度场景使用较低值

#### MaxNeighbors（最大邻居数）

- **作用**: 限制每个实例考虑的邻居数量
- **影响**:
  - 过低：可能遗漏碰撞
  - 过高：性能开销增加
- **建议**: 64（密集场景可增加到 128）

## RVO/ORCA 避障参数

RVO (Reciprocal Velocity Obstacles) 避障系统用于智能路径规划和群体移动。

### 基础参数

| 参数 | 类型 | 默认值 | 范围 | 说明 |
|------|------|--------|------|------|
| bEnableRVO | bool | true | - | 启用RVO避障 |
| NeighborRadius | float | 300 | 100-1000 | RVO邻居检测半径（厘米） |
| TimeHorizon | float | 1.0 | 0.1-5.0 | RVO时间窗（秒） |
| MaxSpeed | float | 0 | 0-2000 | 最大速度，0=使用当前速度 |
| MinSpeed | float | 50 | 0-500 | 最小速度，防止完全停止 |
| ArrivalRadius | float | 200 | 50-500 | 到达目标时的减速半径 |
| ArrivalDensityThreshold | int32 | 6 | 1-20 | 到达密度阈值，防止目标点过度拥挤 |
| MaxNeighbors | int32 | 16 | 4-32 | RVO最大邻居数量 |
| FrameStride | int32 | 1 | 1-4 | RVO分帧步长，2可降低50%计算量 |

### 参数详解

#### NeighborRadius（邻居半径）

- **作用**: 定义避障考虑的邻居范围
- **影响**:
  - 过低：可能无法及时避障
  - 过高：计算量增加，可能过度绕行
- **建议**: 约为碰撞半径的 4-6 倍（200-400 厘米）

#### TimeHorizon（时间窗）

- **作用**: 预测碰撞的时间范围
- **影响**:
  - 过低（0.1-0.5s）：响应迟钝，可能碰撞
  - 过高（2.0-5.0s）：过度避障，绕行距离长
- **建议**: 0.5-1.0 秒

#### MaxSpeed / MinSpeed（速度范围）

- **作用**: 限制实例的移动速度
- **建议**:
  - MaxSpeed = 0：使用实例当前速度
  - MinSpeed > 0：防止实例完全停止

#### ArrivalRadius（到达半径）

- **作用**: 定义"接近目标"的距离
- **影响**: 到达后实例会减速，减少抖动

#### FrameStride（分帧步长）

- **作用**: 每N帧计算一次RVO
- **影响**:
  - 1：每帧计算，最精确
  - 2：每2帧计算，性能提升50%
- **建议**: 大规模场景使用 2

## 抗抖动参数

抗抖动系统用于减少高密度场景中的抖动现象。

### 密度自适应

| 参数 | 类型 | 默认值 | 范围 | 说明 |
|------|------|--------|------|------|
| bEnableDensityAdaptation | bool | true | - | 启用密度自适应 |
| DensityThreshold | int32 | 8 | 1-20 | 触发自适应的邻居数量阈值 |
| HighDensityRelaxationFactor | float | 0.3 | 0.1-0.5 | 高密度时的松弛系数 |

### 速度平滑

| 参数 | 类型 | 默认值 | 范围 | 说明 |
|------|------|--------|------|------|
| bEnableVelocitySmoothing | bool | true | - | 启用速度平滑 |
| VelocitySmoothingFactor | float | 0.15 | 0.05-0.5 | 平滑系数，越大越平滑但响应变慢 |

### 抖动检测

| 参数 | 类型 | 默认值 | 范围 | 说明 |
|------|------|--------|------|------|
| bEnableJitterDetection | bool | true | - | 启用抖动检测 |
| JitterThreshold | float | 0.7 | 0.3-0.9 | 方向变化阈值，超过视为抖动 |
| JitterSuppressionStrength | float | 0.5 | 0.1-1.0 | 抖动抑制强度 |

### 参数详解

#### DensityThreshold（密度阈值）

- **作用**: 定义"高密度"的邻居数量阈值
- **影响**: 超过阈值后，实例会降低响应强度
- **建议**: 6-10

#### VelocitySmoothingFactor（速度平滑系数）

- **作用**: 平滑速度变化，防止突变
- **影响**:
  - 过低（0.05）：平滑效果弱
  - 过高（0.5）：响应迟钝
- **建议**: 0.15-0.25

#### JitterThreshold（抖动阈值）

- **作用**: 检测方向变化的程度
- **影响**:
  - 过低：误判正常移动为抖动
  - 过高：无法检测到抖动
- **建议**: 0.6-0.8（使用点积判断，1.0=同向，-1.0=反向）

## 推荐配置

### 高性能 + 不抖动 + 不重叠

这是最常用的配置，适合大多数场景。

#### PBD 参数

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| 启用PBD碰撞 | true | 开启实例间碰撞 |
| PBD碰撞半径 | 50-80 | 根据角色体型调整，过小会重叠 |
| PBD迭代次数 | 3 | 平衡性能和稳定性，2-4都可以 |
| 障碍后额外迭代 | 1 | 有障碍物时增加稳定性 |
| PBD松弛系数 | 0.3-0.6 | 越低越稳定但响应慢，推荐0.3 |
| PBD网格尺寸倍率 | 1.0 | 保持默认 |
| PBD最大邻居数 | 64 | 密集场景可增加到128 |
| PBD更新频率 | 1 | 每帧更新，保证稳定 |
| 启用速度投影 | true | 防止穿透后抖动 |
| 速度投影强度 | 0.8-1.0 | 越高越稳定 |
| 速度恢复速率 | 2.0 | 碰撞后恢复原速度的速率 |
| 最大推力系数 | 1.5 | 深度重叠时的推力上限 |

#### RVO/ORCA 参数

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| 启用RVO/ORCA | true | 开启智能避障 |
| RVO邻居半径 | 200-400 | 约为碰撞半径的4-6倍 |
| RVO时间窗 | 0.5-1.0s | 越大越提前避障，但可能过度绕行 |
| RVO最大速度 | 0 | 0=使用当前速度 |
| RVO最小速度 | 50-100 | 防止完全停止 |
| 到达半径 | 150-300 | 接近目标时减速范围 |
| 到达密度阈值 | 4-8 | 防止目标点过度拥挤 |
| RVO最大邻居数 | 12-20 | 平衡精度和性能 |
| RVO分帧步长 | 1-2 | 2可降低50%计算量 |

#### 抗抖动配置

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| 启用密度自适应 | true | 高密度时自动降低响应 |
| 密度阈值 | 8 | 邻居数超过此值触发自适应 |
| 高密度松弛系数 | 0.3 | 高密度时的松弛系数 |
| 启用速度平滑 | true | 平滑速度变化防止抖动 |
| 速度平滑系数 | 0.15 | 越大越平滑，但响应变慢 |
| 启用抖动检测 | true | 自动检测并抑制抖动 |
| 抖动检测阈值 | 0.7 | 方向变化超过此值视为抖动 |
| 抖动抑制强度 | 0.5 | 检测到抖动时的抑制力度 |

## 调试建议

### 如果仍有抖动

按优先级调整：

1. **增加 PBD 迭代次数**: 3 → 4
2. **降低 PBD 松弛系数**: 0.6 → 0.3
3. **增加速度平滑系数**: 0.15 → 0.25
4. **降低 RVO 时间窗**: 1.0 → 0.5
5. **增加密度阈值**: 8 → 10

### 如果实例重叠

1. **增加 PBD 碰撞半径**
2. **增加 PBD 迭代次数**: 3 → 5
3. **增加速度投影强度**: 0.8 → 1.0
4. **降低松弛系数**: 0.5 → 0.3

### 如果性能不足

1. **增加 RVO 分帧步长**: 1 → 2
2. **降低最大邻居数**: 64 → 32
3. **降低邻居半径**: 300 → 200
4. **降低 PBD 迭代次数**: 4 → 2

### 使用调试命令

```
Skelot.DrawCollisionRadius  // 显示碰撞半径
Skelot.DrawNeighborLinks    // 显示邻居连接
Skelot.Stats                // 打印统计信息
```
